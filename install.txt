Required software:
  - git
  - cmake
  - C and C++ compilers and MPI wrappers (e.g. openmpi or mpich)

Installation:
  - Choose a "flavor" (e.g. osx-shared, cee-static) that suites your
    environment (or create your own) and edit the install.sh script accordingly.

  - Run the install script in the root directory:
    > bash install.sh
