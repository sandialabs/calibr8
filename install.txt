Required software:
  - git
  - cmake
  - C and C++ compilers and MPI wrappers (e.g. openmpi or mpich)

Installation:
  Run one of the following commands from the root directory:
  - 1) Use a shared library CApp flavor for your OS:
    > bash install.sh

  - 2) Choose a CApp flavor (e.g. flavor=osx-static) and run:
    > bash install.sh $flavor
